## Downloading Forecasts

All forecast files are available in the `model-output/` directory, organized
by team-model. Each CSV file follows the
[submission format](../participate/format.md).

```bash
# Clone the repository
git clone https://github.com/macro-forecast-hub/Macro-Forecast-Hub.git

# Forecasts are in model-output/
ls model-output/
```

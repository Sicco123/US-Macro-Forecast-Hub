# About the Macro Forecast Hub

The Macro Forecast Hub is a collaborative, open-science platform for
probabilistic macroeconomic forecasting.

---

## Motivation

Macroeconomic forecasting is a central activity in economics, finance, and
policy making. Yet most forecasting efforts operate in isolation, making it
difficult to compare methods, combine forecasts, or track performance over time.

The Macro Forecast Hub addresses this by providing:

- A **standardized format** for probabilistic macro forecasts
- **Automated validation** ensuring data quality
- **Rigorous evaluation** using proper scoring rules
- **Open data** for research and reproducibility
- A **hub ensemble** that combines the wisdom of multiple models

---

## Inspiration

This project is inspired by the forecasting hub model developed for
epidemiological surveillance:

- [European COVID-19 Forecast Hub](https://github.com/european-modelling-hubs/RespiCast-Covid19)
- [US COVID-19 Forecast Hub](https://github.com/reichlab/covid19-forecast-hub)
- [Infectious Disease Modeling Hubs](https://hubverse.io/)

We adapt their proven infrastructure — version-controlled submissions,
automated validation, ensemble aggregation, and proper scoring — to the
domain of macroeconomic forecasting.

---

## Goals

1. **Benchmark models**: Provide a level playing field for comparing
   macroeconomic forecasting methods
2. **Improve forecasts**: Leverage forecast combination to produce
   better predictions than any single model
3. **Advance research**: Create a public dataset of real-time probabilistic
   macro forecasts for methodological research
4. **Promote open science**: Make forecasting transparent and reproducible
